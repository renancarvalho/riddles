# Riddles
This repo was created to share riddle solutions using javascript, the main goal is to share knowledge.

###How to send you own riddle ?

Just create a [pull request](http://tableless.com.br/contribuindo-em-projetos-open-source-com-o-github) with the riddle description and rules.

###How to submit your solution for a riddle ?
Just create a [pull request](http://tableless.com.br/contribuindo-em-projetos-open-source-com-o-github) with the riddle solution. Please use jsfiddle links.
